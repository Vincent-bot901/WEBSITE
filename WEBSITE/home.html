Hvordan importeres billedet?

Vi starter med HTML. Først linker vi vores CSS file til vores HTML file, derefter importere vi billedet. Billedet skal har classen "image-cont-1" så at vores billed ser mere asthetisk ud på hjemmesiden. Derefter skal vi i gang med css, vi starter med at skrive vores class til billedet. Vi definer width og height for at definer størlsen af vores billed. Derefter sætter vi border radius til 30px hvilket gøre at billed har rund kanter. Så sætter vi display til block så billedet holder sig til venstre side af hjemmesiden og Margin til right så billedet ikke stå på kanten af hjemmesiden men lidt mere central. Så definer vi også størlsen af vores border og farven. Nu definer vi 2 containers, flex-container er den store container, derfor bruger vi display flex derudover definer vi border radius til at være 30px. Den lille container hvor vi definer størlsen først og derefter position til realative hvilket gøre at den fylder hvor der er plads og som det sidste sætter vi margin til auto som gøre at der plads ved siderne, toppen, og bunden af containeren.
